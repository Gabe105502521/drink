# drink
上癮
請先開啟gps，如未開啟GPS，程式將會crash。  

該程式主要四個功能  

一. 透過列表和random完成隨機決定飲料功能。  

二. 將菜單用記事本儲存，讀取後用hashmap存在各自drinkshop中，以價錢排列後以列表顯示。  

三. 透過google map的search api獲得附近一公里內飲料店的相關資訊，並萃取所要資訊後用以評分及距離排列後列表表示出來  

四. 用列表加選單完成自調飲料的功能。
